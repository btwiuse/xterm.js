Usage

import { Terminal } from '@xterm/xterm';
import { CursorTrailAddon } from 'xterm-addon-cursor-trail';

const term = new Terminal();
term.open(document.getElementById('terminal'));

const trail = new CursorTrailAddon({
  trailColor: '#cccccc',
  trailOpacity: 0.5,
  decayFast: 0.1,
  decaySlow: 0.3,
  startThreshold: 1
});
term.loadAddon(trail);
Trail active — move cursor to see effect
Tip: Move the cursor quickly to see the four-corner elastic chase effect. Leading corners chase fast, trailing corners lag behind.
